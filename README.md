# Faz bem

## Instalando o NodeJS para poder usar o npm:
```
$ curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
$ sudo apt-get install -y nodejs
```

## Instalando A Firebase CLI:
```
$ npm install -g firebase-tools
```

## Inicializar o site:
```
$ firebase init
```

## Para dar o deploy: 
```
$ firebase deploy
```
